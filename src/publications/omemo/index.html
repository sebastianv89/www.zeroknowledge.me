The OMEMO protocol is a Jabber/XMPP implementation of the Signal protocol. The report analyses the cryptographic properties of the specification and conducts a code-review of the reference implementation in the Conversations application for Android.

This assignment was carried out for Radically Open Security and the Pacific Research Alliance.
